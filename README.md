# B2B Detection Guide — Pudeto

Veřejná landing page s průvodcem "Jak detekovat B2B návštěvníky na webu" + možnost stáhnout GTM script.

## Tech stack

- Next.js 14 (App Router)
- TypeScript
- Tailwind CSS
- Statický web bez databáze

## Lokální spuštění

```bash
npm install
npm run dev
```

Web běží na http://localhost:3000

## Nasazení na Vercel

```bash
# Přes Vercel CLI
npm i -g vercel
vercel

# Nebo propoj GitHub repozitář na vercel.com
```
